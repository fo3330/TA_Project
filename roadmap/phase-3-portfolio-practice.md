# Phase 3 — AI+Cloud 결합 실전/포트폴리오 구축

## 목표

- Phase 1·2에서 쌓은 역량을 하나의 실전 프로젝트/포트폴리오로 통합
- 아키텍처 설계 문서를 실제로 작성해보며 TA 직무에 요구되는 산출물 형태에 익숙해지기

## 체크리스트

- [ ] 포트폴리오용 프로젝트 주제 선정 (AI 워크로드를 Cloud 위에 설계·배포하는 형태)
- [ ] 아키텍처 설계 문서(ADR, 시스템 다이어그램 등) 작성 연습
- [ ] 선정 프로젝트를 실제로 클라우드에 배포 (비용/가용성/보안 트레이드오프 문서화)
- [ ] 현재 업무 프로젝트의 AI 활용 경험([docs/current-project-ai-usage.md](../docs/current-project-ai-usage.md))을 포트폴리오 스토리로 재구성
- [ ] 포트폴리오/블로그/GitHub 형태로 외부 공개 가능한 산출물 정리
- [ ] 자격증 취득 진행 (해당 시, [docs/learning-resources.md](../docs/learning-resources.md) 참고)
