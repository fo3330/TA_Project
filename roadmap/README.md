# 로드맵 인덱스

목표: AI + Cloud 환경 설계 중심 Technical Architect(TA)로 이직

이 로드맵은 캘린더 월이 아닌 **단계(Phase)** 기준으로 구성됩니다. 이직 시점이 고정되어 있지 않으므로, 각 Phase는 완료 기준(체크리스트)이 채워지면 다음 단계로 넘어가는 방식으로 진행하세요. 여러 Phase를 동시에 진행해도 무방합니다.

| Phase | 핵심 목표 | 상태 |
|---|---|---|
| [Phase 1](phase-1-cloud-foundations.md) | Cloud 환경 구조 기초 다지기 (최대 취약점) | 진행중 |
| [Phase 2](phase-2-ai-driven-dev.md) | AI를 활용한 개발 역량 강화 | 진행중 |
| [Phase 3](phase-3-portfolio-practice.md) | AI+Cloud 결합 실전/포트폴리오 구축 | 예정 |
| [Phase 4](phase-4-job-transition.md) | 이직 준비 및 실행 | 예정 |

> Phase 1, 2는 취약점 보완이 목적이므로 병행 진행을 권장. Phase 3는 1·2가 어느 정도 궤도에 오른 뒤 시작.
