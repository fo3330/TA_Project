# Phase 2 — AI를 활용한 개발 역량 강화

## 목표

- "AI 앱 개발·배포 초급" 수준에서 벗어나 AI를 실무 도구로 능숙하게 활용
- AI 인프라(모델 서빙, RAG, 에이전트 아키텍처)에 대한 구조적 이해 확보

## 체크리스트

- [ ] AI 코딩 도구(Claude Code 등) 매일 실무/사이드 프로젝트에 사용 습관화
- [ ] LLM 기본 개념 정리 (토큰, 컨텍스트, 임베딩, 파인튜닝 vs 프롬프팅)
- [ ] 간단한 AI 애플리케이션 직접 설계·배포 (API 연동 → 배포까지 1개 이상 완주)
- [ ] RAG(검색증강생성) 파이프라인 구조 학습 및 미니 실습
- [ ] AI 에이전트/멀티에이전트 아키텍처 개념 학습
- [ ] 모델 서빙/추론 인프라(GPU 리소스, 배치 vs 실시간) 개요 학습
- [ ] 학습한 내용을 [docs/current-project-ai-usage.md](../docs/current-project-ai-usage.md)와 연결해 실무 사례로 기록
